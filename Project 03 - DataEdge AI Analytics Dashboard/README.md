# DataEdge AI | Advanced Analytics Dashboard 🌌📊

DataEdge AI is a sophisticated, cloud-inspired data mining and visualization platform. Designed as an "Analytics OS," it allows users to upload datasets and instantly perform high-fidelity exploratory data analysis (EDA), data refining, and statistical modeling through an interactive "Glassmorphism" interface.

## 🚀 Key Features
- **Core Analytics Node**: Instant high-level overview of dataset dimensions, memory payload, and logical structure.
- **Data Refining Studio**: Integrated "Null Scrubber" and "Duplicate Purge" modules to ensure dataset integrity before analysis.
- **Statistical Engine**: Deep topological scans including geometric distribution summaries and interactive correlation heatmaps.
- **Visual Deck**: A high-speed projection interface supporting Bar, Line, Scatter, Area, and 3D-Hole Pie charts with custom grouping.
- **AI Insight Hub**: Automated intelligence nodes that perform synthetic distribution analysis (Mean vs. Median skew detection) and visual violin-plot diagnostics.

## 🛠️ Technology Stack
- **Dashboard Framework**: Streamlit (with Premium Custom CSS overrides).
- **Data Processing**: Python, Pandas, NumPy.
- **Visualization Suite**: Plotly Express, Plotly Graph Objects.
- **UI Design**: Modern "Glassmorphism" with backdrop-filtering and custom-curated Google Fonts (Outfit & JetBrains Mono).

## 📁 Project Structure
- `app.py`: The heart of the Analytics OS containing all 5 intelligence modules.
- `generate_sample.py`: Utility script to create high-quality synthetic data for demonstration.
- `sample_dataset.csv`: Standardized sample data for instant testing.
- `requirements.txt`: Package dependencies for seamless environment setup.

## 🏃 Quick Start

### 1. Install Dependencies
```bash
pip install -r requirements.txt
```

### 2. Launch the Analytics OS
```bash
streamlit run app.py
```
