# VisionEdge AI | Professional Object Detection System 👁️⚡

VisionEdge AI is a high-performance, real-time object detection platform that integrates the state-of-the-art YOLOv8 model with a modern full-stack web architecture. It provides a seamless experience for detecting objects via live webcam feeds, image uploads, and video processing.

## 🚀 Key Features
- **Real-time Neural Scanning**: ultra-low latency webcam detection with bounding boxes and FPS telemetry.
- **YOLOv8 Intelligence**: Powered by Ultralytics YOLOv8 for industry-leading object recognition accuracy.
- **Dual-Mode Processing**: Support for both instant image analysis and live video stream forensics.
- **History Tracking Node**: Locally persists detection events with timestamps and confidence scores.
- **Interactive Confidence Steering**: Real-time slider to adjust detection thresholds on the fly.
- **Glassmorphism Interface**: A high-end dark-themed dashboard built with React and Framer Motion.

## 🛠️ Technology Stack
- **AI Core**: YOLOv8 (Ultralytics), OpenCV, NumPy.
- **Backend Architecture**: Python, Flask, Flask-CORS.
- **Frontend Dashboard**: React.js, Vite, Tailwind CSS, Framer Motion (Animations).
- **Icons**: Lucide Icons.

## 📁 Project Structure
- `backend/`: Flask server containing the YOLO inference engine and live stream logic.
- `frontend/`: React source code for the premium glassmorphism dashboard.
- `samples/`: Sample images for testing the detection engine.
- `run_project.bat`: One-click startup script to initialize both backend and frontend.

## 🏃 Quick Start

### 1. Run the Startup Script
Ensure you have Python and Node.js installed, then simply run:
```bash
./run_project.bat
```
*This will automatically install dependencies and start the servers.*

### 2. Access the Dashboard
- **Frontend**: [http://localhost:3000](http://localhost:3000)
- **Backend API**: [http://localhost:5000](http://localhost:5000)
